Elementos Hijos

Un elemento es hijo de otro elemento si pertenece al contenido de este Ãºltimo.

<!DOCTYPE html>
<html lang="es">
  <head> <!-- Este head es hijo del html. -->
    <!-- Este meta es hijo del head. -->
    <meta charset="UTF-8">
    <!-- Este meta es hijo del head. -->
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <!-- Este title es hijo del head. -->
    <title>Hola Mundo</title>
  </head>
  <body> <!-- Este body es hijo del html. -->
    <!-- Este h1 es hijo del body. -->
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p> <!--Este p es hijo del body.-->
    <!-- Este p es hijo del body. -->
    <p>Este es mi primer ejemplo.</p>
    <footer><!-- Este footer es hijo del body. -->
      <!-- Este p es hijo del footer. -->
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
  </body>
</html>
