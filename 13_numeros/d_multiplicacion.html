En matemáticas

  • (a)(0) = 0

  • (a)(b') = ((a)(b)) + a

  • (2)(0) = 0

  • (2)(1) = ((2)(0)) + 2 = 0 + 2 = 2

  • (2)(2) = ((2)(1)) + 2 = 2 + 2 = 4

  • (2)(3) = ((2)(2)) + 2 = 4 + 2 = 6
