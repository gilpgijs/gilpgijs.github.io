El Elemento img

Representa un hiperenlace y una forma de captura de datos.

El atributo action es el enlace a una URL para procesar su contenido. Puede indicar también procesamiento local con el JavaScript de la página.

Puede manipularse por medio de una colección de elementos asociados.

Algunos de los elementos asociados pueden representar valores editables que se pueden enviar para su procesamiento en un servidor o localmente.

Las formas pueden tener uno o más elementos asociados de tipo <button type="submit"></button>. Como el valor predeterminado de type es submit, se puede escribir <button></button>.

Una introducción al uso de formas está en https://html.spec.whatwg.org/multipage/forms.html#forms

La especificación del tipo de elemento está en https://html.spec.whatwg.org/multipage/forms.html#the-form-element

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Saludo</title>
  </head>
  <body>
    <!--
      El elemento form representa una forma de captura
      de datos.
    
      action="javascript:saluda();"
        Cuando la forma se activa, se ejecuta el
        código de JavaScript
        saluda();
        Con ello se invoca la función saluda que está
        en el elemento <script> que aparece más abajo.
    -->
    <form action="javascript:saluda();">
      <h1>Saludo</h1>
      <p>
        <label>
          Nombre
          <input id="inNombre" type="text" required>
        </label>
      </p>
      <!-- Este button activa la forma. -->
      <p><button>Calcular</button></p>
      <p><output id=outSaludo></output></p>
    </form> <!-- Aquí termina la forma. -->
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      /* Esta es la función que se invoca al activar
       * la forma. */
      function saluda() {
        const nombre = inNombre.value;
        const saludo = `Hola ${nombre}.`;
        outSaludo.value = saludo;
      }
    </script>
  </body>
</html>

Salida.

Maneras de Activar una Forma

  • Haciendo clic en un elemento <button type="submit"></button> asociado con la forma.

  • Usando el tabulador hasta llegar a un elemento <button type="submit"></button> asociado con la forma y oprimir la tecla ESPACIO.

  • Posicionarse en alguno de los campos de captura y oprimir la tecla ENTER.
