Definición del Problema

Antes de empezar a programar, hay que organizar el trabajo, entender el problema y plantear una solución.

Problema

Hacer un programa que muestre al usuario la imagen de un oso. Se presentan 2 opciones para que el usuario seleccione:

Una Rana

Al seleccionar esta opción, el programa le indica que está equivocado.

Un Oso

Al seleccionar esta opción, el programa le indica que está en lo correcto.
