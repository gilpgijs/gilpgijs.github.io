El Elemento html

El elemento html representa una página. Es el elemento principal. Solo puede haber uno de estos elementos en un archivo.

Su especificación está en https://html.spec.whatwg.org/multipage/semantics.html#the-html-element

Tiene 2 elementos hijos, es decir, que pertenecen a su contenido:

head

Define características para procesar el archivo.

Su especificación está en https://html.spec.whatwg.org/multipage/semantics.html#the-head-element

body

El contenido que se muestra.

Su especificación está en https://html.spec.whatwg.org/multipage/sections.html#the-body-element

<!DOCTYPE html>
<!--
  Esta es la etiqueta inicial de los archivos html.
  lang="es"
    lang
      indica el idioma de la página.
    "es"
      español
-->
<html lang="es">
  <!--
    El elemento head define características de la
    página y es uno de los elementos hijos de html.
  -->
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Hola Mundo</title>
  </head><!-- Fin del elemento head. -->
  <!--
    El otro elemento hijo de html es body y define el
    contenido que se muestra.
  -->
  <body>
    <h1>Hola Mundo</h1>
    <p>Hola mundo.</p>
    <p>Este es mi primer ejemplo.</p>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
  </body><!-- Fin del elemento body. -->
</html><!-- Esta es la etiqueta final. -->
