A. Planteamiento del problema

  • Hacer un programa que pida de forma repetida al usuario seleccionar un género musical de una lista de opciones, mientras la opción seleccionada no sea ninguna de las opciones válidas.

C. Pseudocódigo

Algoritmo valida
  Repetir
    Escribir "Selecciona:"
    Escribir "1 Pop"
    Escribir "2 Reguetón"
    Escribir "Introduce número:"
    Leer respuesta
  Hasta Que respuesta = "1" o respuesta = "2"
  Escribir "Adios."
FinAlgoritmo

D. Diagrama de flujo

E. Código fuente

let respuesta = "";
do {
 respuesta = question(
  `Selecciona:
1 - Pop
2 - Reguetón
Introduce número: `)
} while (respuesta !== "1"
 && respuesta !== "2")
console.log("Adios.")
