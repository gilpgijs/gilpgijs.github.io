El Atributo id

El atributo id permite definir un elemento de HTML para ser usando desde JavaScript o css. Se recomienda que su valor siga las reglas de un identificador.

El valor de id no se puede repetir en ningún otro elemento del archivo.

La especificación del atributo está en https://html.spec.whatwg.org/#the-id-attribute

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Chismes</title>
  </head>
  <body>
    <h1>Chismes</h1>
    <p>
      <button onclick="chisme();">
        Mostrar un Chisme
      </button>
    </p>
    <!--
      id="outResultado"
        permite que este output pueda ser usado desde
        el elemento script de más abajo, usando el
        nombre "outResultado".
    -->
    <p><output id="outResultado"></output></p>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      function chisme() {
        /* outResultado se refiere al elemento con
         * id="outResultado"
         * que aparece arriba. */
        outResultado.value =
          "Se tatuó a su novia y lo cortaron.";
      }
    </script>
  </body>
</html>
