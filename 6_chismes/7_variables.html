Variables

Se declaran con let o con var, aunque se recomienda usar let.

Tienen asociado un nombre, que tiene la estructura de un identificador y normalmente empieza con minúscula.

Tienen asociado un valor, que es un dato, el cual puede cambiar durante la ejecución del programa.

El operador de asignación (=) sirve para cambiar el valor de una variable, perdiendo el valor anterior. El operando del lado izquierdo siempre debe ser una variable.

Cuando el nombre de una variable no aparece a la izquierda del operador de asignación, se usa su valor.

Ejemplo

let a = "hola";
let b = "adios";
console.log(a); // Muestra el valor de a (hola).
console.log(b); // Muestra el valor de b (adios).
console.log("a"); // Muestra el texto a.
a = "gg"; // El valor de a cambia a "gg".
console.log(a); // Muestra el valor de a (gg).
b = a; // A b se le asigna el valor de a.
console.log(b);  // Muestra el valor de b (gg).

Salida

hola
adios
a
gg
gg
