El Elemento button

Representa un botón cuya superficie muestra el contenido del elemento.

Cuando el botón se activa, no tiene un comportamiento definido, pero se puede configurar por medio del atributo onclick.

La especificación del tipo de elemento está en https://html.spec.whatwg.org/multipage/form-elements.html#the-button-element

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Chismes</title>
  </head>
  <body>
    <h1>Chismes</h1>
    <p>
      <!--
        Este button muestra en su superficie el texto
          "Mostrar un Chisme".
        onclick="chisme();"
          Al hacer clic en el botón, se ejecuta la
          instrucción
            chisme();
          Con ello, se invoca la función chisme()
          definida en el elemento script más abajo.
      -->
      <button onclick="chisme();">
        Mostrar un Chisme
      </button>
    </p>
    <p><output id="outResultado"></output></p>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      /* Esta function se ejecuta al hacer clic en el
       * button que aparece arriba. */
      function chisme() {
        outResultado.value =
          "Se tatuó a su novia y lo cortaron.";
      } // Al teminar de ejecutarse regresa a onclick.
    </script>
  </body>
</html>
