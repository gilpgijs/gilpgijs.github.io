El Atributo value

Casi cualquier atributo de HTML se puede usar desde JavaScript indicando el id del elemento, seguido de punto (.), seguido del nombre del atributo.

El atributo value solo se puede usar en JavaScript (no en HTML) y representa el texto mostrado por el elemento output.

Al asignar valor al atributo value se cambia el texto que muestra el output.

En este enlace está definido el atributo

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width,
       initial-scale=1.0">
    <title>Chismes</title>
  </head>
  <body>
    <h1>Chismes</h1>
    <p>
      <button onclick="chisme();">
        Mostrar un Chisme
      </button>
    </p>
    <p>
      <output id="outResultado">
      </output>
    </p>
    <footer>
      <p>
        &copy; 2020
        Gilberto Pacheco Gallegos.
      </p>
    </footer>
    <script>
      function chisme() {
        /* Cambia el atributo
         *  "value"
         * del elemento con
         *  id="outResultado",
         * mostrando el texto
         * entre comillas como
         * contenido. */
        outResultado.value =
          "Se tatuó a su novia y lo cortaron.";
      }
    </script>
  </body>
</html>
