Asignación

Aparte del operador de asignación (=), en el caso de los textos hay otro operador que permite cambiar el valor de una variable; la expresión 

a += x;

equivale a

a = a + x;

pero se ejecuta más rápido.

Nota que el primer operando debe ser una variable.

Ejemplo

let c = "h";
const D = ".";
c += "gg"; // c = c + "gg"; c === "hgg"
c += 2; // c = c + 2; c === "hgg2"
c += D; // c = c + D; c === "hgg2."
