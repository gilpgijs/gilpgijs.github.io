Asignación

Aparte del operador de asignación (=), hay otros operadores que permiten cambiar el valor de una variable.

Ejemplo

let b = 4;
let c = "h";
const d = 5;
c += "gg"; // c = c + "gg"; c === "hgg"
c += 2; // c = c + 2; c === "hgg2"
b = d * 2; // b === 10
b += d; // b = b + d; b === 15
b -= 2; // b = b - 2; b === 13
b *= 3; // b = b * 3; b === 39
b /= 3; // b = b / 3; b === 13
b %= 5; // b = b % 5; b === 3
//b **= 2; // b = b ** 2; b === 9
