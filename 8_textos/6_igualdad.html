Operadores de Igualdad

// Igual que (===)
console.log("h" === "gh"); // false
console.log("h" === "H"); // false
console.log("h" === "h"); // true
// Diferente de (!==)
console.log("h" !== "gh"); // true
console.log("h" !== "H"); // true
console.log("h" !== "h"); // false

Ejemplo animado de igual que y diferente de.
