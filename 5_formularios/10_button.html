El Elemento button

Representa un botón cuya superficie muestra el contenido del elemento.

El atributo type controla el comportamiento del botón al ser activado. Sus valores posibles son:

submit

Cuando el botón se activa, el contenido de la forma se envía.

Es el valor por omisión.

reset

Cuando el botón se activa, el contenido de la forma regresa a su estado inicial.

button

Cuando el botón se activa, no tiene un comportamiento definido, pero se puede configurar por medio de un script.

La especificación del tipo de elemento está en https://html.spec.whatwg.org/multipage/form-elements.html#the-button-element

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Saludo</title>
  </head>
  <body>
    <!--
      Cuando un button type="submit" se activa, la
      forma que lo contiene también se activa.
    -->
    <form action="javascript:saluda();">
      <h1>Saludo</h1>
      <p>
        <label>
          Nombre
          <input id="inNombre" type="text" required>
        </label>
      </p>
      <!--
        Este button activa la forma.
        Muestra en su superficie el texto "Calcular".
        Por omisión type="submit".
      -->
      <p><button>Calcular</button></p>
      <p><output id=outSaludo></output></p>
    </form>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      function saluda() {
        const nombre = inNombre.value;
        const saludo = `Hola ${nombre}.`;
        outSaludo.value = saludo;
      }
    </script>
  </body>
</html>
