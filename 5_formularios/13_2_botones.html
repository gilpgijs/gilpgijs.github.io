Formulario con 2 botones

En este ejemplo se capturan números usando input type=number.

Los campos con formato se validan al activar la forma.

Atributos Para Controles Numéricos y de Fecha

min

Valor mínimo que se puede capturar.

max

Valor máximo que se puede capturar.

step

Granularidad o separación entre los valores que se pueden capturar. Por ejemplo, step=0.1 indica que los valores pueden ser 0, 0.1, 0.2, 0.3, etcétera. Aplica también para fechas.

El valor por omisión para controles numéricos es 1.

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Formulario con 2 botones</title>
  </head>
  <body>
    <form action=javascript:calculaSuma();>
      <h1>Formulario con 2 botones</h1>
      <p>
        <label>
          a
          <input id=inA type=number required min="0.1"
            max="100" step="0.1">
        </label>
      </p>
      <p>
        <label>
          b
          <input id=inB type=number required min="0.1"
            max="100" step="0.1">
        </label>
      </p>
      <p>
        <button>Sumar</button>
        <!--
          formaction="javascript:calculaResta();"
              asigna la URL al activar este botón.
              En este caso invola la función
              calculaResta.
        -->
        <button
          formaction="javascript:calculaResta();">
          Restar
        </button>
      </p>
      <p><output id=outResultado></output></p>
    </form>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      function calculaSuma() {
        /* Recupera el número capturado en "inBase" y
         * lo asigna a la constante "base".
         * inA.valueAsNumber
         *   obtiene el número capturado en inA. */
        const a = inA.valueAsNumber;
        const b = inB.valueAsNumber;
        const suma = a + b;
        outResultado.value = `La suma es: ${suma}.`;
      }
      function calculaResta() {
        const a = inA.valueAsNumber;
        const b = inB.valueAsNumber;
        const resta = a - b;
        outResultado.value = `La resta es: ${resta}.`;
      }
    </script>
  </body>
</html>
