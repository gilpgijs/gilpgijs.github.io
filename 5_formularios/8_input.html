El Elemento input

Puede desplegar una gran cantidad de controles según el valor type.

Los campos con formato se validan al activar la forma.

Los valores más usados para type son:

hidden

Campo invisible cuyo valor no puede se examinado o manipulado por el usuario.

text

Manipula texto plano.

search

Usa texto para búsquedas. Se despliega con la apariencia preferida por la plataforma donde se ejecuta.

tel

Texto que representa un número telefónico válido.

url

Texto que representa una URL válida.

email

Texto que representa un email válido.

password

Texto que representa una contraseña y por lo mismo no se muestra.

number

Maneja números.

range

Maneja un rango de números.

En los navegadores móviles, el atributo type muestra un teclado virtual adecuado para el tipo de información que se captura.

Otros Atributos

pattern

Para los tipos que manejan texto.

Expresión regular para validar el contenido. Abarca desde el inicio hasta el final del texto.

La especificación del tipo de elemento está en https://html.spec.whatwg.org/multipage/input.html#the-input-element

      <h1>Saludo</h1>
      <p>
        <label>
          Nombre
          <!--
            Campo de captura de datos.
            id="inNombre"
              identificador del campo de captura. Se
              puede usar desde el script más abajo. El
              id no se puede repetir en la página.
            type="text"
              indica que captura un texto.
            required
              Es un atributo booleano. Cuando aparece
              su valor es true y cuando no, vale
              false. El valor true indica que el campo
              no se puede dejar en blanco cuando se
              active la forma.
          -->
          <input id="inNombre" type="text" required>
        </label>
      </p>
      <p><button>Calcular</button></p>
      <p><output id=outSaludo></output></p>
    </form>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      function saluda() {
        /* La variable inNombre le apunta al campo
         * con id="inNombre" del código HTML.
         * El texto introducido se recupera con
         * inNombre.value */
        const nombre = inNombre.value;
        const saludo = `Hola ${nombre}.`;
        outSaludo.value = saludo;
      }
    </script>
  </body>
