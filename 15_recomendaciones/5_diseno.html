Diseño

Interfaz Gráfica

Se usará una página que presente una página con los siguientes elementos:

Un campo de selección

Para seleccionar el género.

Un botón con el texto Recomendar

Para hacer que se realice el cálculo.

Un elemento output

Para mostrar la recomendación.

Algoritmo

  1. El usuario selecciona un género del campo de selección.

  2. Al hacer clic en el botón Saludar se toma el valor del campo de selección y se almacena en la variable género.

  3. Se obtiene el valor de recomendación

    si género === "", entonces

    recomendación = "Te falta seleccionar un género"

    en otro caso, si género === "balada" o género === "mariachi", entonces

    recomendación = "Juan Gabriel"

    en otro caso, si género === "cumbia" entonces

    recomendación = "Selena"

    en otro caso
    recomendación = "De ese género no conozco.";

  4. Se muestra el valor de recomendación.
