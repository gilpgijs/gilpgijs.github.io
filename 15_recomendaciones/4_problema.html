Definición del Problema

Hacer un programa que recomiende al usuario un artista en función del género que solicite.

Muchos programas realizan procesamiento de datos. Para ello necesitamos identificar:

Datos de Entrada

Los datos que se solicitan al usuario y con los que trabaja el programa.

Datos de Salida

Los datos que produce el programa y que generalmente se obtienen a partir de los datos de entrada.

Procesamiento

Las transformaciones o cálculos que realiza el programa sobre los datos de entrada para obtener los datos de salida.

Datos de Entrada

El género musical.

Datos de Salida

Un artista recomendado.

Procesamiento

Obtener el artista que se recomienda para ciertos géneros.

Escenarios

Son ejemplos con datos para mostrar el comportamiento que debe tener el programa.

Cuando el programa esté listo, se puede probar introduciendo los datos de entrada de cada escenario y el programa debe generar los datos de salida correspondientes.

género = "cumbia"

recomendación = "Selena"

género = "mariachi"

recomendación = "Juan Gabriel"
