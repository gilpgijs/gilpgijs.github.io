El Elemento input

Puede desplegar una gran cantidad de controles, pero usaremos la variante más simple, que es como campo de captura..

La especificación del tipo de elemento está en https://html.spec.whatwg.org/multipage/input.html#the-input-element

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport"
      content="width=device-width, initial-scale=1.0">
    <title>Saludo</title>
  </head>
  <body>
    <h1>Saludo</h1>
    <p>
      <label>
        Nombre
        <!--
          Campo de captura de datos.
          id="inNombre"
            identificador del campo de captura. Se
            puede usar desde el script más abajo. El
            id no se puede repetir en la página.
        -->
        <input id="inNombre">
      </label>
    </p>
    <p>
      <button onclick="saluda();">Saludar</button>
    </p>
    <p><output id=outSaludo></output></p>
    <footer>
      <p>&copy; 2020 Gilberto Pacheco Gallegos.</p>
    </footer>
    <script>
      function saluda() {
        /* La variable inNombre le apunta al campo
         * con id="inNombre" del código HTML.
         * El texto introducido se recupera con
         * inNombre.value */
        const nombre = inNombre.value;
        const saludo = `Hola ${nombre}.`;
        outSaludo.value = saludo;
      }
    </script>
  </body>
</html>

Salida.
