Diseño

Interfaz Gráfica

Se usará una página que presente un formulario con los siguientes elementos:

Un cuadro de texto

Para introducir el nombre de usuario.

Un botón con el texto Saludar

Para hacer que se realice el cálculo.

Un elemento output

Para hacer que se realice el cálculo.

Algoritmo

  1. Al activar la forma, se toma el valor del cuadro de texto y se almacena en la variable nombre.

  2. Se realiza el cálculo

    saludo = `Hola ${nombre}.`.

  3. Se muestra el valor de saludo.
