• También hay operadores de incremento y decremento en 1, pero son algo truculentos; es mejor usarlos solos, sin combinarlos con otros operadores.

Ejemplo

let z = 0
console.log(z) // 0
// Incremento. Aumenta z en 1.
z++
// z === 1
console.log(z) // 1
// Decremento. Disminuye z en 1.
z--
// z === 0
console.log(z) // 0
