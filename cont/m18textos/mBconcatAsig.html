• Equivale a

    a = a + x;

    pero se ejecuta más rápido.

  • Nota que el primer operando debe ser una variable.

Ejemplo

let c = "h"
const D = "."
c += "gg" // c ➝ "hgg"
console.log(c);
c += 2    // c ➝ "hgg2"
console.log(c);
c += D    // c ➝ "hgg2."
console.log(c)

Salida

hgg
hgg2
hgg2.
