Valores Lógicos

Este tipo de datos se llama boolean.

Representa la lógica de valores verdadesos y falsos; también se le conoce como álgebra booleana.

Valores que se Consideran Falsos

  • false

  • Para los operadores lógicos, los valores false, null, undefined, 0 y "" se evalúan a false.

Valores que se Consideran Verdaderos

  • true

  • Para los operadores lógicos, los valores que no sean false, null, undefined, 0 ni "" se evalúan a true.
