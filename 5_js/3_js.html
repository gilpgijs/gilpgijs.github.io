JavaScript

El estándar que define las características de JavaScript y otros lenguajes similares se llama ECMAScript y está definido en https://www.ecma-international.org/publications/standards/Ecma-262.htm.

En 2020, la versión más usada es la edición 6, pero algunos navegadores incorporan caracterísiticas más nuevas.

Para probar los códigos usaremos el sitio JavaScript Tutor en http://www.pythontutor.com/javascript.html.
