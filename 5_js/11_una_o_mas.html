Comentarios de una o más Líneas

Los comentarios de una o más líneas empiezan con el texto /* y terminan con */. Pueden usar una o más líneas.

Ejemplo

/* Muestra "Hola pp". */
console.log("Hola", "pp");
console.log("tt"); /* Muestra "tt". */
/* Muestra
 * "bb". */
console.log("bb");

Salida

Hola pp
tt
bb
